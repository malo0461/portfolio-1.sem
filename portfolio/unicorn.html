Background

The brief was to make an animated story of our own conception using javascript, based on a prepositioned flowchart.

Approach

I reckoned drawing relatively simple characters would be the right approach for this project, as I anticipated JavaScript to be quite time-consuming.

Process

Dramaturgy and initial visual idea was submitted in a hand-in. The main character's psyke was expanded upon with a character sheet, all the characters were sketched by hand and touched up with Adobe Illustrator. Spritesheets were created for each moving character: the unicorn and the mushrooms. The mist was animated using CSS, as were river (1 & 2) and the clouds.

The remaining graphics were drawn with Illustrator and the animation style summed up in a styletile.

The animation was coded in HTML, CSS and JavaScript (jQuery) using Brackets. Sound was edited in Audition and timed in the JavaScript document.

The final animation was documented in a hand-in.
