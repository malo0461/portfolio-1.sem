Portfolio

Inspiration right out the window

A seemingly dreary view of Frederikssundsvej turned out to be a nice template for my portfolio layout.

Playing with wireframes

I decided to start with the desktop layout and not take the mobile first approach, since my site primarily would be viewed on a wider screen. I did, however, take into account that my layout should be responsive.

After experimenting with different layouts, I decided to work with no. 3.

Process

I have coded this site in Brackets using HTML, CSS and JavaScript (jQuery). I used a code snippet found on stackoverflow.com for inspiration when making the 'back' arrow on the portfolio sub-pages.

For the scroll reveal function on this page I have used a code snippet found on JSFiddle as a template, and modified it to fit my content.

For the colour overlay on the portfolio page I used an example from W3 Schools for inspiration.
